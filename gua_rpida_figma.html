Guía Rápida Figma

Figma Web

1. Crear tu primer lienzo

Una vez que entres en figma.com y te registres:

  1. Haz clic en el botón "+ Design file" arriba a la derecha.

  2. Se abrirá una interfaz limpia. Lo primero que debes hacer es presionar la tecla F (o seleccionar el icono de rejilla en la barra superior).

  3. Verás una lista de tamaños predefinidos a la derecha (iPhone, Desktop, etc.). Selecciona uno para crear tu Frame (el área de trabajo).

2. Herramientas básicas de dibujo

En la barra superior tienes todo lo que necesitas. Los atajos son tus mejores amigos:

  • V (Move): Para seleccionar y mover objetos.

  • R (Rectangle): Para crear cajas y botones.

  • O (Ellipse): Para círculos o avatares.

  • T (Text): Para escribir. Haz clic en el lienzo y escribe lo que quieras.

3. El panel de Diseño (Derecha)

Aquí es donde ocurre la magia visual. Cuando seleccionas un objeto, este panel cambia para mostrarte:

  • Fill (Relleno): Cambia el color del objeto.

  • Stroke (Trazo): Añade bordes.

  • Effects: Para añadir sombras (Drop shadow) o desenfoques (Layer blur).

  • Alignment: Los iconos de arriba sirven para centrar objetos automáticamente.

4. El "Súper Poder" de Figma: Auto Layout

Si vas a diseñar interfaces (UI), esto es obligatorio. Permite que tus botones o listas crezcan según el contenido:

  1. Selecciona un texto y un rectángulo (o dos objetos).

  2. Presiona Shift + A.

  3. Ahora los objetos están "atados" y puedes ajustar el espaciado entre ellos de forma exacta en el panel derecho.

5. Capas y Páginas (Izquierda)

En el panel izquierdo verás la lista de todo lo que has dibujado.

  • Layers: Las capas de arriba tapan a las de abajo.

  • Pages: Puedes crear diferentes páginas (ej. "Bocetos" y "Diseño Final") para mantener el orden.

6. Prototipado y Compartir

¿Quieres ver cómo se vería en un celular real?

  1. Cambia la pestaña de Design a Prototype (arriba a la derecha).

  2. Haz clic en un objeto y arrastra el círculo azul que aparece hacia otro Frame para conectarlos.

  3. Dale al icono de "Play" (Present) arriba a la derecha para ver la simulación.

Resumen de atajos rápidos para empezar:

Acción Atajo
Moverse por el lienzo Mantener Barra Espaciadora + Arrastrar
Zoom In / Out Ctrl / Cmd + Rueda del ratón
Duplicar objeto Mantener Alt + Arrastrar
Agrupar objetos Ctrl / Cmd + G
