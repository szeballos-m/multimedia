Plantilla del Manual Técnico

Estructura del Manual Técnico Profesional

  1. Portada y Ficha Técnica
  • Nombre del Proyecto: (Ej: "InfoMuseo: Guía Interactiva").
  • Versión del Software: (Ej: v1.0.2).
  • Fecha de entrega:
  • Autor/es:
  1. Especificaciones de Hardware (Requerimientos)

Este apartado define dónde "vive" el sistema.

  • Plataforma de destino: (Ej: Windows 11, Totem Android, macOS).
  • Requisitos mínimos: Procesador, memoria RAM y espacio en disco.
  • Periféricos necesarios: (Ej: Pantalla táctil 1920x1080, altavoces, lector de QR).
  1. Arquitectura del Sistema
  • Software de Autoría: Herramienta utilizada (ej: Adobe Animate, Hype, etc.).
  • Mapa de Navegación Final: El diagrama definitivo que muestra cómo están conectados todos los nodos.
  • Inventario de Activos (Assets): Tabla con el formato y resolución de los medios principales (Videos .mp4, Audios .mp3, etc.).
  1. Guía de Instalación y Ejecución
  • Proceso de Compilación: Cómo se generó el ejecutable.
  • Instalación: Pasos para copiar el sistema al entorno final.
  • Modo Kiosco: Instrucciones para que el sistema se inicie automáticamente al encender la computadora y el usuario no pueda "cerrar" la aplicación para ver el escritorio.
  1. Guía de Mantenimiento y Solución de Problemas (Troubleshooting)
  • Errores comunes: ¿Qué hacer si el video no carga o si el botón de inicio no responde?
  • Actualización de contenidos: Si el cliente quisiera cambiar un texto o una imagen en el futuro, ¿dónde están guardados los archivos fuente?
  1. Pruebas de Calidad (QA)
  • Resumen del reporte de bugs que realizaron en la Unidad III.1.
  • Listado de correcciones finales aplicadas antes del empaquetado.
