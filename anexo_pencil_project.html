ANEXO Pencil Project

Guía para usar Pencil Project

1. Instalación y apertura

  • Descarga Pencil Project desde su sitio oficial (es gratuito y portable).
  • Ábrelo: verás una interfaz con un área de trabajo central y una barra lateral con elementos de interfaz (llamados stencils).

2. Crear un nuevo proyecto

  • Ve a File → New Document.
  • Asigna un nombre a tu proyecto (ej. WireframesMultimedial).
  • Puedes trabajar en varias páginas dentro del mismo documento (cada página puede representar una pantalla).

3. Wireframes de baja fidelidad

  • Arrastra formas básicas (rectángulos, líneas, texto) desde la barra lateral al área de trabajo.
  • Representa menús, botones y áreas de contenido de manera simple.
  • No uses colores ni estilos elaborados: lo importante es la estructura y jerarquía.
  • Ejemplo: un rectángulo grande para “pantalla principal”, dentro rectángulos más pequeños para “menú” y “contenido”.

4. Wireframes de media fidelidad

  • Usa los stencils prediseñados que trae Pencil (botones, menús desplegables, campos de texto, íconos).
  • Ajusta tipografías básicas y tamaños de texto para diferenciar títulos, subtítulos y contenido.
  • Representa interacciones: añade flechas o notas que indiquen qué pasa al hacer clic.
  • Mantén un diseño más cercano a la interfaz real, pero sin llegar al acabado gráfico final.

5. Organización del flujo

  • Crea una página por cada pantalla principal (Inicio, Menú, Galería, Configuración, etc.).
  • Usa flechas o anotaciones para mostrar cómo se conectan entre sí.
  • Esto te permitirá visualizar el flujo de usuario junto con los wireframes.

6. Exportar y guardar

  • Guarda tu trabajo en formato .epgz (archivo editable de Pencil).
  • Para entregar o incluir en tu informe, ve a File → Export Document y selecciona PNG o PDF.
  • Así tendrás imágenes claras de tus wireframes para incluir en tu práctico.

ü  Consejos útiles

  • Empieza siempre con baja fidelidad para definir la estructura.
  • Luego pasa a media fidelidad para mostrar detalles de interacción.
  • Usa etiquetas claras en cada elemento (ej. “Botón: reproducir”, “Menú: configuración”).
  • Mantén consistencia en los símbolos y estilos entre pantallas.
